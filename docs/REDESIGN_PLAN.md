# Polymarket-Style UI Redesign Plan

## Design System
- **Colors:**
  - Dark BG: #0f0f23
  - Card BG: #1a1a2e
  - Card Hover: #252541
  - Border: #2d2d44
  - Purple Primary: #7c3aed
  - Blue Accent: #3b82f6
  - Green (Positive): #10b981
  - Red (Negative): #ef4444
  - Text: #e5e5f0
  - Muted Text: #9ca3af

## Pages to Redesign
1. **Login Page (/)** - Dark theme, centered card, purple gradient button
2. **Squads List (/squads)** - Card grid layout like Polymarket events
3. **Squad Chat (/squads/[id])** - Split view: chat left, leaderboard right
4. **Profile (/profile)** - Settings card with purple accents

## Key Changes
- Remove light mode completely (Polymarket is dark-only)
- Use card-based layouts with hover effects
- Purple gradient buttons for primary actions
- Clean, minimal borders
- Market-like feel for leaderboard
